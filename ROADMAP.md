# Benflux UI — Roadmap

## Phase 1: Foundation (v0.1 — Current)

- [x] Monorepo setup (TurboRepo + pnpm)
- [x] Core tooling (TypeScript, ESLint, Prettier, Tailwind)
- [x] `@benflux-ui/utils` — cn, cva, format utilities
- [x] `@benflux-ui/hooks` — useMediaQuery, useDebounce, useLocalStorage, useInView...
- [x] `@benflux-ui/themes` — 8 themes (light, dark, amoled, glass, neon, cyberpunk, luxury, minimal)
- [x] `@benflux-ui/animations` — Framer Motion presets
- [x] `@benflux-ui/react` — Foundation components
  - [x] Button (9 variants)
  - [x] Input, Textarea
  - [x] Card (6 variants), Badge, Avatar, AvatarGroup
  - [x] Dialog, Toast (Sonner), Skeleton, Progress
  - [x] Tabs (4 variants), Command/CommandDialog
  - [x] CodeBlock, ChatUI
- [x] WOW Effects: AuroraBackground, Meteors, MagicCard, Spotlight, Marquee, AnimatedBeam, AnimatedText, BorderBeam, ParticleSystem, BentoGrid
- [x] `benflux-ui` CLI — init, add, theme, list
- [x] Documentation site (Next.js 15)
- [x] GitHub Actions CI/CD + Changesets release
